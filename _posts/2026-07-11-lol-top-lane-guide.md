## 第一章  上单位置理解与核心思维

### 1.1 上单的角色定位

上单是一条独立的1v1线，也是全图最长的一条单人线。你的核心任务是：在前期建立个人优势，中期通过TP支援或分带影响地图，后期成为团队的前排支柱或侧翼威胁。

上单玩家必须具备以下三种核心能力：

- **对线压制力** — 知道什么时候该压、什么时候该怂
- **兵线理解力** — 能精准控制兵线位置，让对手难受
- **地图阅读力** — 3秒看一次小地图是基本功

### 1.2 上单英雄分类

| 类型 | 代表英雄 | 核心打法 | 优劣势 |
|------|----------|----------|--------|
| 战士/重装 | 剑姬、诺手、奥拉夫 | 线上压制+中期分带 | 对线强，怕被风筝 |
| 坦克 | 奥恩、塞恩、石头人 | 抗压发育+后期开团 | 团战强，对线弱 |
| 法师/AP | 凯南、吸血鬼、格温 | 发育到关键装备发力 | 后期强，前期弱 |
| 长手压制 | 奎因、薇恩、杰斯 | 点人压制+推塔节奏 | 被抓必死，团战难发挥 |

### 1.3 超凡大师上单的思维模式

1. 永远思考「我这一波做了什么能让团队赢」——而不是「我能不能单杀对面」
2. 建立「对方打野位置」的第六感。如果你3秒没看小地图，你已经失位了
3. 每一波兵线都有目的——要么推进、要么回推、要么冻结
4. 被单杀不可怕，可怕的是不知道为什么会死
5. 上单不是孤岛，10分钟前你的TP就是全队的第二条命

## 第二章  BP阶段——赢在起跑线

### 2.1 理解BP的本质

BP（Ban/Pick）不是禁掉你不喜欢的英雄，而是一场信息博弈。作为上单，你要在BP阶段完成三件事：

- **保护自己** — Ban掉克制你英雄池的Counter位
- **限制对方** — Ban掉版本T0上单或者对面绝活哥
- **欺骗对手** — 用选人顺序迷惑对方，不让对方轻易拿到Counter位

### 2.2 当前版本（S15）上单Ban位优先级

| 优先级 | 英雄 | Ban的理由 | 应对方法（如果不Ban） |
|--------|------|-----------|----------------------|
| T0 | 剑姬、诺手 | 单线统治力极强，低分段稳定Carry | 选石头人/奥恩抗压，叫打野帮2次即可 |
| T1 | 贾克斯、鳄鱼 | 中期曲线太强，E技能克制普攻类上单 | 长手英雄风筝，或选AP英雄规避E |
| T2 | 兰博、凯南 | 团战AOE爆炸，线上压制力强 | 选强开团阵容，直接集火秒杀 |

### 2.3 Counter Pick 核心逻辑

BP阶段的终极武器是Counter Pick（后选优势）。如果你在蓝色方（先选），要做好被Counter的心理准备。

- 远程克制近战（奎因克制诺手）
- 高爆发克制持续输出（鳄鱼克制剑魔）
- 真实伤害克制坦克（剑姬克制奥恩）
- 强回复克制消耗（狼人克制杰斯）
- 坦度克制刺客/爆发（石头人克制瑞文）

### 2.4 BP实战流程演示

假设你在蓝色方，以下是标准的上单BP流程：

1. **Ban阶段**：秒Ban剑姬（版本T0），或者Ban你最怕的英雄
2. **观察对方Ban**：如果对方Ban了你的绝活英雄（如剑姬），说明对方上单大概率怕这个
3. **Pick阶段**：蓝色方先选，选一个稳定性高的英雄（如奥恩、鳄鱼），不要一抢需要Counter位的英雄
4. **等待Counter**：红色方上单会后选Counter你，你提前想好被Counter后的打法

### 2.5 看阵容选英雄

BP不只看对线，更要看团队阵容。你的英雄必须满足团队需求：

- 团队缺前排 → 选坦克（奥恩、塞恩）
- 团队缺AP伤害 → 选AP上单（凯南、格温）
- 团队缺分带 → 选战士（剑姬、卡蜜尔）
- 团队缺开团 → 选团控（石头人、奥恩）

## 第三章  符文与召唤师技能搭配

### 3.1 符文系统核心理解

符文决定你对线期的强度和发力时间点。不要抄OP.GG就完事，要根据对线英雄微调。

### 3.2 针对不同对线的符文微调

| 对线类型 | 主符文建议 | 副系建议 | 原因 |
|----------|-----------|----------|------|
| 优势对线 | 征服者/致命节奏 | 坚决系（骸骨+复苏） | 增加对拼和换血收益 |
| 劣势对线 | 不灭之握/迅捷步法 | 启迪系（饼干+扭曲药水） | 增强续航和抗压能力 |
| 坦克对线 | 不灭之握 | 精密系（凯旋+欢欣） | 坦克换血靠不灭消耗 |
| 长手对线 | 迅捷步法/不灭之握 | 坚决系（复苏之风+过度生长） | 增强回复，顶住消耗 |

### 3.3 召唤师技能选择

- **闪现+TP** — 标准选择。TP是联动地图的最强工具
- **闪现+点燃** — 线上压制型（剑姬、诺手）。牺牲支援换取对线击杀能力
- **闪现+疾跑** — 对灵活性要求高的英雄（贾克斯、剑魔）
- **闪现+虚弱** — 对抗爆发型英雄（对抗瑞文时）

### 3.4 天赋碎片（小符文）微调

- 对抗AD英雄 → 护甲 + 护甲 + 生命值
- 对抗AP英雄 → 魔抗 + 魔抗 + 生命值
- 混合对线 → 护甲 + 魔抗 + 生命值

> **核心原则**：小符文要根据对线实际伤害类型来调整。对抗兰博，护甲碎片是废的，必须带魔抗。这是白金到大师的分水岭。

## 第四章  对线期核心技巧（1-6级）

### 4.1 一级团与上线时机

一级团是赌博，不是战术。作为上单，你的首要任务不是一级团，而是准时上线获取兵线控制权。

1. 游戏1分30秒，兵线出生。1分50秒，兵线交汇。你必须在1分50秒之前到线
2. 帮打野开野最多用3下平A+一个技能，然后立刻走人，绝不恋战
3. 如果对方打野是入侵型（盲僧、豹女），你需要帮打野做好河道视野再上线

### 4.2 1-3级：前三个近战兵决定一切

前三个近战兵的经验归属，决定了谁能先到2级。先到2级的一方多一个技能，多一套伤害，多一段血量。

1. 如果你能先到2：立刻前压，至少逼出对方一瓶药或一个召唤师技能
2. 如果对方先到2：立刻后撤，退到己方近战兵线后方，绝不硬拼
3. 永远记住：1级对2级，等于用刀对枪——你打不赢的

### 4.3 3-5级：建立优势的关键期

3级时，大部分上单的完整技能组已经就位（QWER中的3个技能）。这是第一次真正意义的博弈——谁能先打出有效换血，谁就能在后续5分钟掌握主动权。

- **在对方补刀僵直时出手** — 观察对方近战兵血量，在对方将要补刀时攻击他。他要么放弃这个刀，要么吃你一套伤害再补刀。两种结果你都赚
- **技能不要无脑交完** — 留一个位移/控制技能用于撤退。多数上单死亡的根源是「交完所有技能后被人反打」
- **做好防Gank视野** — 3分15秒是打野清完另一半野区的时间点，这时必须插眼

### 4.4 6级大招：质变时机

6级是大招的质变点。不同英雄的6级强度差异极大，必须心中有数。

| 6级强度 | 英雄举例 | 策略 |
|---------|----------|------|
| 质变级 | 诺手、剑姬、鳄鱼、石头人 | 到6立刻找机会开大击杀，注意对方打野位置 |
| 增强级 | 贾克斯、奥恩、卡蜜尔 | R提升对拼能力，但不要强行越塔 |
| 平稳级 | 杰斯、船长、纳尔 | 6级前就有大招，不要硬等到6再发力 |
| 无作用级 | 奎因（游走型R） | 到6后反而不利对拼，要利用R去支援其他路 |

### 4.5 对线期节奏总结

- 1-2级：抢2、建立血量优势
- 3-5级：换血、建立兵线控制
- 6级：根据英雄强度选择击杀或继续压制

## 第五章  兵线管理学

### 5.1 兵线是上单的第二个血条

兵线控制（Wave Management）是区分普通上单和优秀上单的核心技能。一名超凡大师级上单可以让对手在被控线时生不如死——什么也做不了，只能看着你把兵线冻结在塔前。

### 5.2 兵线四种基本形态

| 形态 | 如何形成 | 目的与效果 |
|------|----------|------------|
| **推进（Push）** | 用技能/AOE快速清掉对方远程兵，让你的近战兵多于对方 | 把兵线推到对方塔下，让对方在塔下漏刀，为你创造回家或游走窗口 |
| **回推（Slow Push）** | 只补最后一刀，不清远程兵，让己方兵线慢慢积攒 | 积攒一大波兵线推进，配合打野越塔的完美时机 |
| **冻结（Freeze）** | 保持对方远程兵比己方多3-4个，只补最后一刀，不推线 | 把兵线锁在己方塔前，对方无法补刀，还要被Gank。这是上单最强的折磨手段 |
| **重置（Reset）** | 把兵线完全推进对方塔下，让新一波兵在中路交汇 | 打破对方控线，重置兵线位置，为新的控线周期做准备 |

### 5.3 冻结兵线的详细操作

冻结是最重要也最难掌握的兵线技术。一旦你学会冻结，对方上单会生不如死。

**冻结的三个必要条件：**

- 对方远程兵数量必须多于你的远程兵（至少多3个）
- 兵线位置在己方半区（靠近你的塔但不在塔的攻击范围内）
- 你不能有任何AOE技能打到远程兵

**操作步骤：**

1. 推线进塔后等兵线回弹，或者引诱对方推你的线
2. 当兵线来到己方半区时，观察双方远程兵数量
3. 若对方远程兵 ≥ 己方远程兵+3，开始「只补最后一刀，绝不用AOE」
4. 用身体挡在对方英雄和远程兵之间，阻止对方推线进塔
5. 维持冻结直到你需要回家、对方打野来了、或者想推进了

### 5.4 兵线决策流程

| 如果... | 那么... |
|---------|---------|
| 你比对方强（装备/等级领先） | 冻结兵线在塔前 → 逼对方出来补刀 → 单杀或逼闪 |
| 你比对方弱（被Counter/劣势） | 尽量让兵线进塔 → 安全补刀 → 等打野来帮 |
| 你想回家更新装备 | 推线进对方塔下 → 回家 → 走回线上（不交TP） |
| 打野在附近想抓你 | 放线到己方塔下 → 配合打野反打 → 击杀 |
| 己方打野在附近想Gank | 放线或回推 → 勾引对方前压 → 配合打野击杀 |
| 小龙即将刷新 | 推线 → 回家更新装备 → TP到下路 → 5v4打小龙团 |

### 5.5 炮车兵线的特殊处理

每三波会出现一波炮车兵。炮车兵波是关键的时间节点：

- 炮车波推线速度更快 — 如果你想回家，选择炮车波推线效率最高
- 对方推炮车波进塔，你会在塔下漏刀 — 必要时用技能补炮车
- 炮车波是越塔的最佳时机 — 因为炮车可以多抗2-3下防御塔

## 第六章  换血与单杀时机判断

### 6.1 换血的底层逻辑

换血不是「我打你一套你也打我一套」，而是「我用最小的代价换取你最大的损失」。一次成功的换血应该满足以下至少两点：

1. 你有兵线优势（你的小兵更多）— 对方打你会拉到小兵仇恨，承受额外伤害
2. 对方在补刀僵直（攻击前摇动画）中 — 对方无法反击
3. 对方关键技能在CD — 例如诺手的Q刚用过、剑姬的W刚交完
4. 你有多兰盾/药水等恢复优势 — 你可以慢慢回血，对方不能

### 6.2 换血的核心技巧

**补刀僵直截击** — 当对方近战兵血量低于对方普攻斩杀线时，对方会往前走一步去补刀。就在他攻击动作开始的一瞬间，你出手。他有0.3-0.5秒无法还击，这段时间你白打他一套。这是上单最基本的换血基本功。

**技能CD窗口期** — 每个英雄的技能都有CD。诺手Q的CD在1级是9秒，剑姬Q是13秒，鳄鱼E是18秒。记住对方核心技能的CD，在他的技能真空期上前换血。高手过招，全在CD计算。

**草丛消仇恨** — 当你攻击对方英雄时，对方小兵会转而攻击你。进入草丛可以瞬间消除小兵的仇恨。因此，换血后第一时间进入草丛，可以让对方小兵停止攻击你，而你的小兵仍在攻击对方。这就是「草丛换血」的精髓。

**塔下消耗** — 如果你把对方压在塔下，可以在对方补塔刀时用远程技能消耗。但要记住——防御塔会攻击你如果你造成伤害。所以：只交瞬间伤害的技能（如剑魔Q1段），然后立刻离开塔的攻击范围。

### 6.3 击杀时机的判断

什么时候可以「ALL IN」追求击杀？满足以下三个条件越多越好：

1. 对方血量低于40%，且无闪现/屏障等保命技能
2. 你的大招已就绪，且大招是你核心的斩杀技能（诺手R、剑姬R等）
3. 对方打野在小地图上被定位在中下路（至少10秒赶不到上路）
4. 兵线在己方半区 — 对方逃跑路线更长

**击杀前检查清单：**
- 检查对方是否有点燃/虚弱等反打召唤师技能
- 检查对方R技能的CD（按Tab看大招绿点）
- 看小地图确认打野位置 — 这是最重要的，一次成功击杀被对方打野收掉毫无意义

### 6.4 不要为了击杀而击杀

超凡大师和钻石分段最大的区别之一：大师知道什么时候不该杀人。

- 对方残血缩塔，你没有越塔能力 — 不杀，推线进塔让塔吃掉他的兵
- 击杀后你只剩100血，兵线停在对方塔前 — 等于送了一个TP或回程窗口给对方
- 你击杀对方但被对方打野赶来收掉 — 1换1在上路不赚，因为你可能掉两波兵+塔皮

## 第七章  TP使用艺术

### 7.1 TP不是回线工具

超凡大师以下的上单，90%把TP用于回线（死亡后传送回线上或者回家后TP回线）。这是最大的浪费。TP是联盟最强的地图影响力工具，你的第一次TP可以决定整场比赛的走向。

### 7.2 TP的四种正确用途

| 用途 | 时机 | 执行细节 |
|------|------|----------|
| **下路支援** | 下路发生2v2/3v3团战时，TP到下路参与战斗 | TP草丛眼位或后排小兵。你1个人创造3v2的人数优势 |
| **小龙/先锋团** | 资源刷新前30秒推线，然后回家更新装备TP回来 | 先手推线获取线权，回家更新核心装备，TP到龙坑参与团战 |
| **保塔/守线** | 己方一塔残血且被推线时，TP到防御塔保护塔皮 | 保住塔皮=160-320金币+经验，同时防止对方推塔后转线 |
| **包夹/绕后** | 己方开起团战后，TP到对方后方/侧面 | TP到对方撤退路线上的眼位，形成前后夹击。TP绕后是上单的终极姿态 |

### 7.3 TP时机判断流程

当你看到下路打起来时，问自己三个问题：

1. 对方上单有TP吗？如果他有，你们同时TP下去就是4v4，单纯的资源互换
2. 我这波TP能稳定拿到至少一个击杀或一条龙吗？如果不能，别T
3. 上路兵线处理好没有？如果兵线在你塔下而你TP走了，掉两波兵+3层塔皮，亏到爆炸

### 7.4 TP落地位置选择

- 后排小兵 — 最安全，不容易被断
- 草丛眼 — 对方没视野时，可以打对面一个出其不意
- 防御塔 — 最保守，用于守线或保塔
- 对方撤退路线的眼位 — 用于绕后包夹

### 7.5 不带TP时的替代策略

如果你带了点燃而不是TP（剑姬、诺手），你需要做到：

- 用线上压制力弥补没有TP的劣势 — 你必须打穿对面上单
- 提前用走动支援 — 提前10秒往战场走，而不是等打起来了再走
- 推线快回家快 — 利用兵线窗口期快速回家再走回线上

## 第八章  中期转线与资源团

### 8.1 14分钟前：一塔的重要争夺

14分钟前推掉一塔会获得额外的塔皮金币（160g/层，共5层）。上单一塔的争夺，本质是兵线权和经济权之争。

1. 拿到对面一塔 = 800塔皮金币 + 400塔金币 + 兵线自由权
2. 被推掉一塔 = 失去800经济 + 进入防守模式 + 对方上单可以自由游走
3. 塔皮的争夺在7分30秒左右进入关键期 — 第5层之后的塔皮就是纯赚

### 8.2 14分钟后：解放上路

推掉一塔后，你在上路的活动空间扩大了整整一倍。这时候你需要做选择：

- **继续分带** — 适用于剑姬、贾克斯、卡蜜尔等分带英雄。持续推进二塔，对方必须派人来守，队友以4v3团战
- **转线支援** — 适用于奥恩、石头人、凯南等团战英雄。推完一塔后直接转中线，帮助中单推塔
- **换到下路** — 适用于下路一塔也被推掉后。换到下路继续推进，利用TP参与上半区资源争夺

### 8.3 先锋与大龙的控制

上单是离先锋最近的人，先锋的控制与你息息相关。

- 8分钟先锋刷新 — 提前1分钟（7分钟）开始推线，获取线权
- 拥有线权的上单可以优先移动到先锋位置，配合打野拿下
- 如果对方上单没有线权，他要么掉兵经验来争先锋（亏），要么放弃先锋（更亏）
- 拿到先锋后，用来撞一塔扩大优势。不要用来撞满血塔 — 等塔皮快掉完了再用

### 8.4 中期团战站位原则

中期团战中，你的定位取决于你的英雄类型：

- **坦克** — 站在队伍最前方，为C位吃伤害
- **战士** — 侧翼站位，找机会切入对方后排
- **AP** — 后排站位，等待对方聚集后释放AOE大招
- **分带英雄** — 不在团战中！你在另一路推进

## 第九章  分带体系执行

### 9.1 分带的核心原理

41分带（1人分带，4人抱团）是上单带领队伍获胜的经典体系。分带的目标不是你自己玩得开心，而是利用边路压力撕裂对方防线。

分带的三个核心要素：

1. 你对对方守线英雄有单杀能力或推线优势 — 这是分带的前提
2. 你的队友在另一侧施加压力 — 让他们不得不分人去守你
3. 你有视野知道你什么时候该退 — 分带最怕的不是打不过，而是被抓死

### 9.2 分带英雄排行榜

| 评级 | 英雄 | 分带优势 |
|------|------|----------|
| **S级** | 剑姬、卡蜜尔 | 1v1几乎无解，推塔极快，逃生能力强 |
| **A级** | 贾克斯、剑魔、蛮王 | 单挑强力，推塔快，有一定逃生能力 |
| **B级** | 纳尔、杰斯、船长 | 推线快但1v1能力一般，需要视野保护 |
| **不适合分带** | 奥恩、石头人、塞恩 | 推塔慢、1v1弱、逃生差。这些英雄的使命是跟团开团 |

### 9.3 分带节奏控制

分带的节奏控制是一门精巧的艺术。

- **对方2人来抓你而队友4v3推进** → 拖时间，不要死。你有逃生技能，利用草丛和视野周旋
- **对方1人来守你且你打得过** → 压到塔下，消耗塔血量。逼对方叫更多人
- **对方5人在中路逼团** → 快速判断：你能推掉二塔/高地吗？如果能，继续推；如果不能，TP支援
- **对方消失且没有视野** → 立刻后退。分带最忌讳的是「贪一波」，被抓死等于送优势

### 9.4 边路视野布置

分带必须有视野支持。没有视野的分带是在送人头。

- 在对方野区入口放置控制守卫 — 看到对方来抓就立刻撤退
- 利用占卜花朵排清对方视野，让对方不知道你的位置
- 如果对方没有你的视野，他们必须派至少2个人来找你 — 这本身就创造了空间

## 第十章  后期团战与终结比赛

### 10.1 后期上单的使命

30分钟后，比赛进入后期。一波团战的胜负直接决定大龙/远古龙归属，进而决定比赛结果。后期上单的使命非常明确：在团战中发挥最大作用，带领团队走向胜利。

### 10.2 团战前的准备工作

1. 检查核心装备 — 你是否已经出到了关键装备（血手、复活甲、自然之力等）
2. 确认召唤师技能状态 — 闪现差10秒不要打团，等10秒
3. 提前布置视野 — 大龙/远古龙刷新前40秒，己方必须有2个以上的龙区视野
4. 处理好兵线 — 团战前确保边路兵线是推过去的。团赢一波团但没兵线推塔=白赢

### 10.3 不同上单的团战打法

**战士/重装（剑姬、诺手、剑魔）：**
- 不要第一个进场！等对方交了1-2个关键控制后再切入
- 目标永远是对方ADC和中单，不要和对面坦克纠缠
- 剑姬R触发回血阵是大团的神技 — 盯住最容易触发弱点的目标R
- 诺手的核心是触发血怒 — 找到残血目标用R斩杀，触发5层血怒后你就是神

**坦克（奥恩、石头人、塞恩）：**
- 你是团队的第一道防线，也是第一把尖刀
- 奥恩R是远程开团神技 — 判断对方站位密集时释放
- 石头人R的目标不是对方前排，而是对方ADC和AP — 撞到C位你就赢了
- 开团后立刻使用控制链限制对方C位输出，为队友创造输出环境

**AP（凯南、兰博、格温）：**
- 你的大招是团战胜负的关键 — 必须等到对方站位密集或者交了位移技能后再放
- 凯南闪现R进场是最经典的开团方式 — 但必须确认队友能跟上
- 兰博R封住对方撤退路线或逼迫对方走进你的阵型
- 格温W是免伤神技 — 在W范围内你的生存能力翻倍

**长手（杰斯、奎因）：**
- 后期你是第二个AD射手，站后排输出
- 杰斯POKE团 — 团战前用加强Q消耗对方血量，降到半血再开
- 奎因利用R的机动性，绕后找到对方落单C位击杀

### 10.4 远古龙与大龙的决策

| 场景 | 正确决策 |
|------|----------|
| 你方5人都在，对方少1人 | 直接开龙。5v4是最完美的开龙窗口 |
| 双方满员对峙 | 不要强行开龙。先消耗，等对方失误或有人走位靠前 |
| 远古龙刷新，大龙也在 | 远古龙优先。远古龙的斩杀效果远超纳什男爵 |
| 团灭对方但己方只剩1-2人 | 优先推高地塔。能推掉一路高地比打一条龙更值 |
| 团灭对方且己方有4-5人存活 | 优先终结比赛。如果推不了一路，再考虑大龙 |

### 10.5 终结比赛的最后一击

拿到大龙后，利用大龙BUFF推进是终结比赛的标准方式。

1. 大龙BUFF持续180秒，你有充裕的时间慢慢推进。不要着急，越急越容易失误
2. 131分带是最强的大龙推进阵型 — 上单带一路、中单带一路、下路+打野+辅助推中路
3. 大龙BUFF强化的小兵在大炮车身上。保护炮车，让它磨塔的血量
4. 对方会殊死一搏开团 — 保持阵型，不要被开。让对方先着急
5. 推掉门牙塔后，不要堵泉水 — 直接攻击水晶，胜利才是最重要的

### 10.6 心态管理

从超凡大师冲向最强王者，最后10%的差距在心态。

- 队友失误时：打信号，不要打字。打字不能赢比赛，信号可以
- 自己被单杀后：立刻调整心态，思考接下来10分钟怎么打回来。一把游戏53%的胜率在你手里
- 对面全屏嘲讽时：/mute all。安静打游戏的上单才是最强的上单
- 劣势局不要点投降：一局30分钟的比赛，有无数次翻盘的机会。上单是翻盘能力最强的位置
- 连胜后不要骄傲、连败后不要气馁：保持每局的专注度一致，这是通往王者之路的必修课

## 终章

**大道至简，上单之道在于克制与耐心。**

上单是联盟中最孤独也最考验个人能力的位置。你拥有最长的单人线，最充裕的发育空间，也承担着最大的责任。一个优秀的上单可以让对方在你的塔前绝望，可以让对方的ADC在你的绕后中颤抖。

记住这几句话：

1. 对线不是全部，但一切从对线开始
2. 兵线是你最强大的武器
3. TP不是为了回线，是为了改变比赛
4. 被单杀不可怕，不知道原因才可怕
5. 你的冷静，就是团队的底气

—— 一位超凡大师上单玩家
